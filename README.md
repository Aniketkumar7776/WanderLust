# WANDERLUST

An Airbnb like Website

## FULL STACK PROJECT

- Frontend(HTML, CSS, Javascript)
- Backend(Express)
- Database(MongoDB)
- Browse property listings
- Create, edit and delete listings

## Additional Features

- User Authentication & Authorization
- Mongoose
- Express Sessions
- REST APIs & CRUD Operation integration
- Features of Reviews & Ratings
- Image upload support(Cloudinary)

## Future Enhancements

- Payment integration
- Completely working Search box
- Working of different categories when clicked
